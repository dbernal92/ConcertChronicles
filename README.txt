SBA-307: HTML/CSS
Concert Chronicles Website

Description:
This project is a multi-page website built using HTML, CSS, and Flexbox. It showcases a fictional website called "Concert Chronicles," dedicated to concert enthusiasts. Users can explore concert highlights, tips, favorite performances, and interact with forms for subscriptions and inquiries.

Features:
1. Three Pages:
   - Home Page: Highlights recent concerts with images and text.
   - Tips Page: Provides concert preparation tips and additional multimedia elements.
   - Contact Page: Allows users to send messages and subscribe to updates.

2. Content Types:
   - Text, images, and GIFs

3. HTML Tags

4. CSS Styling:
   - Inline, internal, and external styling is used.
   - Flexbox is utilized for layout consistency and responsiveness.
   - CSS animations provide interactive and engaging visuals.

5. Forms:
   - Subscription form on the Home page.
   - Contact form on the Contact page.

6. Dropdown Menu:
   - Includes frequency selection for subscription preferences.

7. Table:
   - Displays concert information.

Technical Requirements:
- HTML Requirements:
  - Contains at least three pages.
  - Maintains a consistent grid system.
  - Uses at least ten unique HTML tags.
  - Includes text, images, and GIFs.
  - Includes two forms and a dropdown menu.

-CSS Requirements:
  - Inline, internal, and external styling.
  - Five different CSS selectors.
  - Complementary color palette.
  - Flexbox used for layout.
  - Two CSS animations.

How to Run:
1. Clone or download the project files.
2. Open the `index.html` file in your web browser.
3. Navigate through the pages via the navigation bar.
